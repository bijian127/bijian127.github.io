**时间**：周四 20:00 | **字数**：2000-2500字 | **形式**：深度长文 + 代码截图

**场景**：你每周要读 10+ 份行业报告 PDF，每份 30-50 页。全部读完不现实，但你又不想错过关键信息。

**效果预览**：
- Before：每份报告花 30 分钟快速浏览，10 份 = 5 小时，还是只看了个大概
- After：Python 脚本自动调用 AI API，10 份报告 5 分钟全部出摘要

**操作步骤**：
1. 安装 Python 和 DeepSeek API（DeepSeek API 免费额度足够个人用）
2. 复制下方 5 行代码，保存为 summarize_pdfs.py
3. 把所有 PDF 放到一个文件夹，运行脚本
4. 等待 5 分钟，打开 output 文件夹，每份 PDF 的摘要已经生成

```python
import os, requests
from pypdf import PdfReader

API_KEY = "你的DeepSeek API Key"
folder = "./reports"  # 放PDF的文件夹

for f in os.listdir(folder):
    if f.endswith(".pdf"):
        text = "".join(p.extract_text() or "" for p in PdfReader(os.path.join(folder, f)).pages)
        r = requests.post("https://api.deepseek.com/chat/completions",
            headers={"Authorization": f"Bearer {API_KEY}"},
            json={"model":"deepseek-chat","messages":[{"role":"user","content":f"请用300字以内总结这份报告的核心观点：\n{text[:8000]}"}]})
        with open(f"./output/{f}.txt","w") as out:
            out.write(r.json()["choices"][0]["message"]["content"])
        print(f"Done: {f}")
```

**前置准备**：安装依赖 `pip install pypdf requests --break-system-packages`。DeepSeek API 在 platform.deepseek.com 免费注册。
