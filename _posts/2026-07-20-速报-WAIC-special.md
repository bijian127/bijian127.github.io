## 1. DeepSeek V4 正式版本周发布，峰谷定价详解

如果你一直在用 DeepSeek，这条是今天最重要的消息：**V4 正式版（GA）最快本周发布**。灰度测试已全面铺开，部分用户已拿到权限。

### 1.1 两个版本，全系百万上下文

| | **V4 Pro（旗舰）** | **V4 Flash（轻量）** |
|---|---|---|
| 总参数 | 1.6 万亿 | 2840 亿 |
| 激活参数 | 490 亿 | 130 亿 |
| 上下文窗口 | 100 万 token | 100 万 token |
| 架构 | MoE（混合专家） | MoE（混合专家） |
| 定位 | 重度任务（代码、Agent、长文） | 日常轻量调用 |

> 100 万 token 是什么概念？整本《三体》（约 90 万字）可以一次性丢进去，不用分段。

### 1.2 首创峰谷定价，详解怎么省钱

DeepSeek 这次引入了类似电力行业的「削峰填谷」计费模式。**高峰时段价格翻倍，平时段维持低价**：

**时段划分**：
- **高峰时段**（工作日 9:00-12:00、14:00-18:00，每天 7 小时）
- **平峰时段**（12:00-14:00 + 18:00-次日 9:00 + 周末/节假日，每天 17 小时）

**完整定价表**（单位：元/百万 tokens）：

| 计费项 | V4 Pro 平时 | V4 Pro 高峰 | V4 Flash 平时 | V4 Flash 高峰 |
|--------|------------|------------|-------------|-------------|
| 输入（缓存命中） | 0.025 | 0.05 | 0.02 | 0.04 |
| 输入（缓存未命中） | 3 | 6 | 1 | 2 |
| 输出 | 6 | 12 | 2 | 4 |

![DeepSeek V4 发布概念图](/assets/images/illustration-deepseek-v4.jpg)
*DeepSeek V4 正式版发布，百万上下文 + 峰谷定价，开源模型新标杆*

**关键结论**：
- V4 Flash 缓存命中的输入价格仅为 **0.02 元/百万 token**，几乎等于免费
- 高峰时段翻倍后，V4 Pro 输出 12 元/百万 token，仍仅为 Claude Opus 4.6（约 525 元）的 **1/44**
- 即使是 V4 Pro 高峰价格，也仅为 GPT-5.6 Sol 的大约 **1/17**

### 1.3 性能实测数据

| 基准测试 | V4 Pro 得分 | 对比参考 |
|---------|-----------|---------|
| SWE-bench（软件工程） | **80.6%** | Claude Opus 4.6: 80.8%（几乎持平） |
| LiveCodeBench Pass@1 | **93.5** | 对比组最高 |
| Codeforces 评分 | **3206** | 人类选手排名第 23 位 |
| Agent 能力 | **1554** | 开源权重模型第一 |

开发者实测案例：用 V4 生成一个完整 3D 游戏，**总花费仅 0.7 元**。

### 1.4 核心技术突破

V4 并非简单"参数变大"，而是架构层面的创新：

- **混合注意力架构**：结合 CSA（压缩稀疏注意力）+ HCA（重度压缩注意力），百万 token 场景下推理 FLOPs 仅为 V3.2 的 **27%**，KV 缓存空间降至 **10%**
- **流形约束超连接（mHC）**：增强跨层信号传播，提升模型稳定性
- **Muon 优化器**：更快的收敛速度，万亿参数级训练更高效

### 1.5 跟你有什么关系？

- 如果你在用 DeepSeek API 做自动化（PDF 批量摘要、邮件自动回复等），**等 V4 GA 上线后迁移过去**，成本还能再降一大截
- 百万上下文全系标配，意味着你可以把整个代码库、整份年报、整本书一次性丢给 AI
- 建议把非实时任务（批量数据处理、模型微调）安排在**平峰时段（晚上/周末）**，成本减半

### 2.2 从「能不能聊」到「能不能干」

前两届 WAIC 还在比参数、刷榜。今年，所有人都在问一个更现实的问题：**能不能用？能不能量产？能不能赚钱？**

Kimi 在开幕前发布了 2.8 万亿参数的 K3 模型，但比参数更值得关注的是 Kimi 同时强调了 Agent 场景下的专项优化——**参数不再是目的，而是为"智能体能干活"提供基座**。

- **腾讯**：混元 Hy3 调用量较前代增长 **68 倍**，登顶 OpenRouter 全球大模型调用量总榜。发布 **WorkBuddy 独立 App**（iOS/Android/鸿蒙三端同时上线），实测可将团队行政效率提升 20%+
- **阿里**：发布企业级 AI 应用创作平台「**秒悟团队版**」（Meoo Team），支持统一身份管理、额度管控、团队资产共享。同时预告 **Qwen3.8**（参数规模达 2.4 万亿，称能力仅次于 Anthropic Fable 5）
- **蚂蚁**：推出 **Agentar 2.0** 平台，内置 200 套财务、法务、供应链等岗位模板，企业无需程序员即可一键生成专属 AI 员工
- **商汤**：发布 **SenseNova U1 Pro**，定位面向长程任务的交付级原生多模态智能体基座

IDC 在大会上预判：**2026 年全球将新增近 5000 万个 AI 数字员工集中上岗**。

### 2.3 具身智能元年：机器人开始上班

具身智能首次被列为 WAIC 核心赛道，**242 家**机器人与智能硬件企业参展，占全部展商的 23.6%。

关键变化：往年机器人展区是跳舞、翻跟头，今年变成了**工厂、仓库、药房的实景部署**：

- **智元机器人**：精灵 G2 Max 与京东物流合作的仓储部署场景实景呈现——这是人形机器人**首次在真实仓储作业中落地部署**。智元在会场公共区域部署了 60 台机器人做指路引导
- **傅利叶**：发布首款高负载轮式双臂机器人 GRW，最高 16 公斤稳定负载
- **节卡机器人**：JAKA K1-25 双臂合计额定负载 50kg，填补国产重载人形机器人空白

### 2.4 算力竞争逻辑重构：从单颗芯片到超节点

华为 **Atlas 950 超节点**真机首秀：16 台计算柜、上千张算力卡、1024 颗 AI 芯片，总算力达每秒 100-200 亿亿次浮点运算。华为、中兴、曙光、燧原、沐曦、摩尔线程等纷纷亮出超节点产品。

> **一句话总结**：大模型没有消失，它正在像电力、芯片一样被嵌入更庞大的技术架构。AI 行业从"技术驱动"正式转向"应用驱动"。

## 3. Kimi K3 上线两天「卖断货」：暂停新用户订阅

Kimi K3（2.8 万亿参数，100 万 token 上下文）7 月 16 日上线后，需求暴涨到令人咋舌的程度。

### 3.1 发生了什么？

7 月 19 日深夜，月之暗面发布官方公告：

> "过去 48 小时，用户请求量已大幅超出我们的预估，并且**逼近现有集群的承载极限**。"

宣布**即日起暂停 C 端新用户订阅**，全部算力优先保障已订阅用户。

### 3.2 为什么这么火？

K3 的核心参数和定价：

| 指标 | 数据 |
|-----|------|
| 总参数 | 2.8 万亿（全球最大开源模型） |
| 上下文 | 100 万 token |
| 输入价格（缓存命中） | 2 元/百万 token |
| 输入价格（缓存未命中） | 20 元/百万 token |
| 输出价格 | 100 元/百万 token |

Vercel CEO Guillermo Rauch 评测称：**K3 兼顾召回率、精度和价格，GPT-5.6 Sol 效果更强但成本超 K3 七倍**。

Kimi 总裁张予彤 7 月 18 日在朋友圈透露，K3 发布次日（7 月 17 日）ARR（年化收入）创下**历史最大单日增幅**。彭博社称这打破了"美国 AI 领先中国"的固有认知，华尔街出现「Kimi 时刻」讨论，费半指数单日下跌 20%。

### 3.3 后续计划

- 已在全速推进算力扩容，新算力到位后将**逐步开放更多订阅名额**
- 后续会员将拆分为「**Kimi 主权益**」（Web + App + Work）和「**Kimi Code 权益**」（编程工作流专用），以更精准匹配算力

- 评估器在**客户端运行**，代码不离开客户基础设施
